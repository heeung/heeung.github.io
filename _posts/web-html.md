# Web - html

HTML : Hyper Text Markup Language

- 하이퍼 텍스트
    
    하이퍼 텍스트(Hypertext)는 하이퍼 링크를 이용해 독자가 원하는 순서에 따라 기존 문서에서 다른 문서로 접근할 수 있는 텍스트.
    
    즉, 링크라는 통로를 통해 원하는 페이지로 바로 도착하는 접근 방식을 가지고 있음.
    
- 마크업 언어
    
    마크업(Markup) 이란 어떠한 표시를 뜻하며, 마크업 언어(Markup Language)는 태그를 이용하여 문서나 데이터의 구조를 명시하는 언어를 지칭한다.
    

### <!DOCTYPE html>

문서 형식(Document type)을 정의할 때 사용하는 문장, HTML 문서의 가장 처음에 배치되어 이 문서가 HTML 문서라고 브라우저에 알려주는 역할을 수행함.

### <html lang=”en”>

: `<html>`태그는 전체 HTML 문서를 감싸는 태그, 단 한 번 사용할 수 있음.

`<html>`태그 바깥에는 다른 태그가 존재할 수 없다.

`lang=”en”`은 이 페이지의 주 언어가 영어라는 것을 명시함.

### <head>

: 일반 사용자는 웹 사이트에서 `<head>`태그에 속한 정보를 볼 수 없다. `<head>`태그 또한 HTML문서 안에 단 하나 존재할 수 있다.

- <meta> : 문서와 관련된 정보를 담는 태그.
- <title> : 웹 페이지의 제목을 담는 태그.

### <body>

: HTML 문서 중 사용자에게 실제로 보여지는 부분. HTML 문서 안에 단 하나 존재할 수 있다.

- <h1> : 제목을 표시하는 태그. 중요도가 1 ~ 6 까지 있음. 1이 제일 높음.
- <p> : 단락을 표시하는 태그, 컨텐츠 중 글, 본문에 해당하는 내용을 포함합니다. (웹 구조에서 단락을 나눠버림, 안나누는 것도 있음)

### 시맨틱 태그(Semantic tag)

개발자들이 선호하는 레이아웃을 HTML5에서 받아들여 아예 태그로 만들어서 각 요소의 쓰임을 명확하게 구분하였다.

시맨틱 태그는 의미를 가지는 태그로, 이름만으로도 이해할 수 있다.

![Screen Shot 2022-11-22 at 2.21.50 PM.png](https://s3-us-west-2.amazonaws.com/secure.notion-static.com/a1069bf2-c5f6-4a8d-88a3-7fd453db72df/Screen_Shot_2022-11-22_at_2.21.50_PM.png)

- <header> : 웹 페이지 혹은 <section>의 소개나 제목을 담기 위해 사용하는 요소
- <nav> : 내비게이션 역할을 수행하는 요소로, 페이지 이동을 위한 메뉴를 주로 담는다.
- <section> : 기준에 따라 구획을 구분하기 위해 하용하는 요소.
- <article> : 주 내용을 담기 위해 사용하는 요소
- <aside> : 광고 또는 사이트의 주변 부분에 해당하는 내용을 담기 위해 사용하는 요소
- <footer> : 일반적으로 웹 사이트의 가장 아래에 들어가는 회사 정보나 사이트 정보 등의 추가 정보를 담기 위해 사용하는 요소.
